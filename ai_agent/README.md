# 🤖 Wagtail Starter Kit Maintenance Agent

## 📌 Overview

This AI agent helps developers maintain and improve Wagtail starter kit projects by analyzing project structure, setup configuration, and best practices.

---

## 🚀 Features

- Checks for Wagtail dependency
- Detects missing setup scripts (Makefile, setup.bat)
- Validates project structure
- Ensures README presence
- Provides improvement suggestions

---

## ⚙️ Usage

Run from project root:

```bash
python ai_agent/agent.py
